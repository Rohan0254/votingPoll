## Setup instructions

Use Below Command To Clone Repo.
- git clone https://github.com/Rohan0254/votingPoll.git

Configure Database
- Create Database named 'votingpole'

Now Migrate Table use below command
- php artisan migrate

Run App
- php artisan serve
